# Gator 🐊

Gator is a terminal-based RSS feed aggregator built with:

- Node.js
- TypeScript
- PostgreSQL
- Drizzle ORM

It allows users to:

- Register and login
- Add RSS feeds
- Follow and unfollow feeds
- Store posts in a database
- Browse the latest posts from followed feeds

---

## 🚀 Requirements

Before running gator, make sure you have:

- Node.js v20+
- PostgreSQL running locally
- npm installed
